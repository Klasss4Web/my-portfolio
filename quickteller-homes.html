Quickteller Homes Website

Quickteller Homes is a platform designed to bring property owners and customers together in one platform. It allows property owners upload their properties and customers a powerful search feature to search for properties of their choice either Rent, Sales or Shortlet.

This app is owned by Interswitch Group, from initial conception to building and maintenance. I am currently working as a frontend Engineer for this platform. My task includes converting figma designs to actual user interfaces using Reactjs and SCSS for styling. Other tasks include API fetching process. The UI built using React/Redux and Chakra UI as the CSS library.
