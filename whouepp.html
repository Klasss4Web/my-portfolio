whoUepp App

whoUepp P2P Lending App

whoUepp is a peer to peer lending platform where people can become lenders and borrowers. Lenders can set their rates, while the admin matches lenders to borrowers

whoUepp is is a Product of Promena Services Limited. Zealight innovation labs built the product. I led the frontend team that built the application using React and context api for state management. Building of the UI, connecting of the end points and deploying the frontend app to netlify test server. It has an official website, an admin and mobile app. The app has two versions, the regular app and the enterprise appication
