SUA DSEP Cohort 6

SUA Cohort 6 Registration

Skillup Africa website is the official website for the registration of prospective students for the Skillup Africa Digital Empowerment Program(DSEP) Cohort 6. It shows all information about DSEP and it also serves as the platform for registration with paystack payment integrated for collection of registration fee.

This website is owned by Skillup Africa, built by Zealight innovation labs. I was the frontend lead during this project and our tasks include building the user interface of this app. I was also actively involved in the API fetching process. The UI built using React/Context API and Chakra UI as the CSS library.
