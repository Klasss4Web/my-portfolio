Enrolled Agents Website

Registered Accountants in the USA

Enrolled Agent is a platform for registered accountants resident in the United States. People with needs of a qualified accuntant can access the portal, view list of registered accountants and their ratings and book for their services.

This project was executed by BamyX Technologies, As part of the front end team that worked on this project, I contributed in building part of the user interfaces using HTML, CSS, Sass, EJS and JavaScript.

One of the challenges I had working in this project was working in a remote team for the very first time, with time I adjusted with the help of team members. Some of my tasks include building part of the user interface using HTML, Sass, and EJS. I also attended daily progress report meetings and also did regular push to the project gitlab repository.
