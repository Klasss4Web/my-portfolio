GitHub Users

GitHub users search engine

This app is used to search for any user on github without going through github website. User data were collected via github's graphql API.

One of my first project that involved fetching of API was an ajax request I made to query github REST API. When I heard about graphql, I felt I should do something with it, so I replicated the project I did that involed querying github's REST API, but this time I querried github's graphql API and I also added a bit of functionality, to make the experience not far away from what a user would get from github's page.

This page was built with a mobile first approach and its fully responsive across different screen sizes. The goal of this project on the long run is to integrate multiple social media platform in one app.
