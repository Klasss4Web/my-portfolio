LeadCapital Logistics

Delivering Excellence in Every Shipment

Lead Capital Logistics is a logistics platform. It allows individuals and cooporate individuals to register as partners and bring in their vehicles to deliver logistics services and get paid while the organisation takes a commision for their platform.

This project was executed by Zealight Innovation Labs, As the front end lead that worked on this project, I led the team to building the user interfaces, optimizing the application performance and building reusable components.

One of the challenges I had working in this project was having to deal with changing features based on client's requirements and also delivering these updates on time. Some of my tasks include building part of the user interface using React, assigning tasks, demoing of the UI, code reviews and mentorship. I also attended daily progress report meetings and also did regular push to the project gitlab repository.
