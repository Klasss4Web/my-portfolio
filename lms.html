LMS App

Lms Web App

Learning Management System or LMS is an online learning platform where people can register and checkout courses and start learning. It is made for both learners and instructors. I worked with a team of front end developers and UI/UX designers to build the front end with HTML and Bootstrap

LMS is built with HTML on the frontend. The styling was done with Bootstrap library and a bit of pure CSS. The backend was done with Laravel. LMS is built by Zealight innovation labs.
