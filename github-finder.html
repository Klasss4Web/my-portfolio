Github Finder

Github finder is a simple and responsive web app built with html css and javascript. It is basically used to search for github users with their username

When I first heard of github's open sourced graphql api, I felt I should work with it to strenghten my graphql api fetching skills. This app allows you to search for github users with their user names, see their profiles and view their repositories.
