Napollo Website

Napollo is a fictitious and static website built from a random ui I came across on the internet. It was built using React and Bootstrap.

This single page website represent a fictitious music streaming website. The aim of the exercise was to connect the website to an API in order to bring the app alive.

Building the UI and making it responsive was quite easy but I encountered a challenge with the logo which was not available, i didnt want to change it, so i decided to build the entire logo with pure CSS.
