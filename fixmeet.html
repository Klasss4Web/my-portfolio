Fixmeet App

Fixmeet Web App

Fixmeet is a polling app used to allow participants vote on their prefarred date and time for a particular meeting to hold. Its responsive and fits across all device sizes. I handled the frontend(building the UI and fetching the APIs) of this app while i collaborated with the product design and backend teams.

Fixmeet is built with React on the frontend. The styling was done with Chakra UI library and a bit of pure CSS. State management was done with Redux and the backend was done with Laravel. Its a free app built by Zealight innovation labs.
