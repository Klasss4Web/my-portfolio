Estate App PWA

v PWA App

Estate app is used for access control in a gated estate. This pwa is one piece of the larger access control system, its used by residents to invite guests and pay estate dues

This app is owned by Zealight innovation labs. I worked with front end team to build the user interface of this app. I was also actively involved in the API fetching process. The UI built using React/Redux and Chakra UI as the CSS library.
